#### Finanza

Maneja tus finanzas de manera simple con este programa

Estructura Transacciones:
Puedes encontrarlo en transaction.json
```
{
    "amount": 0,
    "type": "EXPENSES"
}
```

Las Transacciones se guardan en localStorage. Cada Transaccion tiene un identificador
Formato de identificador:
EJ: tx1
**tx** -> Es el identificador de transaccion
**1** -> Es el numero de transaccion
